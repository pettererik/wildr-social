Privacy Policy — Wildr Social Publishing

Last updated: 17 July 2026

This privacy policy covers the wildr-social application, an internal publishing tool operated by Wildr (wildr.shop) that posts content to Wildr's own Facebook Page and Instagram account via Meta's official APIs.

What data the app handles

The app is used exclusively by Wildr's own team to publish Wildr's own marketing content (photos, videos and captions). It accesses only the Facebook Page and Instagram professional account owned by Wildr, using access tokens granted by the page administrator.

What data the app does NOT collect

  • It does not collect, store or process any personal data from visitors, followers or any other Facebook/Instagram users.
  • It does not read messages, comments, follower lists or profile information.
  • It does not use cookies, tracking or analytics.

Data storage

Published content (images and captions) is stored in a GitHub repository controlled by Wildr. Access tokens are stored as encrypted secrets in GitHub Actions and are never shared with third parties.

Data deletion

The app holds no user data, so there is nothing to delete. If you have any questions or requests, contact us and we will respond promptly.
